# Contributing

1. Fork and clone the **repo**.

2. To install the client dependencies

   ```javascript
      yarn client:install
   ```

3. To start running the server, Run

   ```javascript
      yarn start:client
   ```

4. To install the server dependencies, Run

   ```javascript
      yarn server:install
   ```

5. To start running the backend server, Run

   ```javascript
      yarn start:server
   ```
